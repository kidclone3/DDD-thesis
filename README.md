# DDD-thesis

## Note some tricks

1. If `biber` failed to run (check the `$? != 0`), clean the cache by 
    ```
    rm -rf `biber --cache`
    ``` 
    and re-run the `biber` command.


2. I use this web [Latex_table_web](https://www.tablesgenerator.com/latex_tables) to generate Latex tables from csv.
